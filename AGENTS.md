# Instructions for AI Agents

- Do not use floating-point arithmetic or any other imprecise calculations during written arithmetic operations.
- Zoom interactions on the window should only apply to the main canvas.
- When zooming out causes the canvas to become smaller than the viewport, it should lock to the center.
- When panning at any zoom level, the canvas should never move in a direction that would take it out of view — it must stop at the edges.
- After a resize, centering and clamping must be recalculated correctly.
- Step-by-step explanations during written arithmetic should be written so that a third-grade elementary school student can understand them.
